# Интерактивный атлас зарубежной литературы

## Как открыть

1. Откройте файл `index.html` двойным щелчком.
2. Интернет не нужен для карты, ленты времени и тестов.
3. Ссылки на источники цитат открываются через интернет.
4. Результаты сохраняются в браузере через `localStorage`.

## Что включено

- интерактивная карта мира;
- карточки авторов и литературных памятников;
- проблематика ключевых произведений;
- русский исследователь для каждого автора;
- лента времени с выделением современников;
- восемь режимов тренировки;
- игры «цитата — адресат» и «цитата — автор»;
- отдельные подборки по реализму, Маннам, Воннегуту и Томасу Вулфу;
- 100 вопросов программы МГПУ и 5 дополнительных вопросов о Томасе Вулфе;
- генератор билета из двух литературных вопросов;
- отдельные отметки «Знаю» и «Не знаю»;
- повышенная вероятность повторения слабых вопросов;
- отдельная страница о Томасе Вулфе и романе «Взгляни на дом свой, Ангел».

## Как разместить в интернете

Папку можно загрузить на GitHub Pages, Netlify, Timeweb, Beget или другой статический хостинг. Серверная часть не требуется.

## Где менять данные

- авторы, проблематика, исследователи, цитаты и вопросы: `data.js`;
- логика тестов, билетов и страниц: `app.js`;
- внешний вид: `styles.css`.

Картографическая основа создана из Natural Earth low resolution. Данные Natural Earth находятся в public domain.
